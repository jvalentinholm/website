# Design System

**Project:** Atlas
**Version:** 1.0
**Status:** Approved
**Date:** 2026-07-06

---

# Purpose

This document defines the visual and interaction principles for all Circle X digital products.

The objective is to create a recognisable, scalable and timeless digital identity.

---

# Design Philosophy

Industrial.

Confident.

Calm.

Scandinavian.

Timeless.

Every visual element should support clarity rather than decoration.

---

# Brand Personality

Circle X should feel:

- Industrial
- Credible
- Intelligent
- Ambitious
- Human
- Innovative

Never:

- Loud
- Aggressive
- Trend-driven
- Over-designed

---

# Visual Principles

Large typography.

Generous whitespace.

Strong hierarchy.

Minimal colour palette.

Illustrations before decoration.

Photography only when authentic.

---

# Colour Principles

Primary colours should communicate trust and industrial quality.

Accent colours should only guide attention.

Colours must never compete with the content.

---

# Typography Principles

Readable before expressive.

Large headlines.

Comfortable reading width.

Consistent spacing.

---

# Illustration Principles

Illustrations explain.

They never decorate.

Illustrations should simplify industrial concepts.

Avoid overly technical engineering drawings.

---

# Photography Principles

Use real facilities.

Use real employees.

Use real products.

Avoid stock photography.

Avoid AI-generated industrial imagery.

---

# Interaction Principles

Motion should support understanding.

Animations should feel calm.

Interactions should feel effortless.

Never animate for entertainment.

---

# Accessibility

Accessibility is a design requirement.

Not an afterthought.

---

# Success Criteria

Every page should communicate:

Industrial quality.

Technical credibility.

Commercial maturity.
