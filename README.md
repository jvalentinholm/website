# Circle X Website

Project Atlas v0.4 — Next.js frontend for Circle X Group.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Design direction

Nordic industrial design, based on Circle X design manual colors:

- Core Green: #265D37
- Fresh Green: #5FB130
- Orange: #E59515
- Off-white: #EFE3CE
- Petroleum: #326875
- Black: #231F20
